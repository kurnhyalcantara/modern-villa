# Authentication & Security

## Authentication

- Email/password login
- Google OAuth login
- Secure session management

## Authorization

- Role-based access control
- User role
- Admin role

## Security Requirements

### Validation

- Zod validation everywhere
- Server-side validation mandatory

### Protection

- CSRF protection
- XSS protection
- SQL injection prevention

### API Security

- Rate limiting
- Request validation
- Auth middleware

### Secrets

- Environment variables must never be exposed
- Use .env.local

### Passwords

- bcrypt hashing

### File Uploads

- Validate file type
- Validate file size

### Session Security

- HTTP-only cookies
- Secure cookies
