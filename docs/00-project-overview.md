# Modern Villa Booking Platform

## Overview

A modern villa booking platform built with Next.js 15, Supabase, Prisma, and Tailwind CSS.

The platform supports:

- Villa booking
- User wallet system
- Deposit & withdrawal
- Reviews & ratings
- Admin management
- Real-time availability
- Secure transactional booking flow

---

## Core Features

### User Features

- Register/Login
- Google OAuth
- Search villas
- Book villas
- Deposit balance
- Withdraw balance
- Review villas
- Notification system

### Admin Features

- Manage villas
- Manage users
- Manage transactions
- Revenue analytics
- Review moderation
